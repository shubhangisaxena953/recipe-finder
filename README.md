# Recipe Finder App
A Vue.js-based web application that allows users to search for recipes based on available ingredients using the Spoonacular API. Users can save favorite recipes, generate shopping lists, and manage their data securely with Firebase authentication and Firestore.

## Features
- **Ingredient-Based Search:** Enter ingredients to find matching recipes with details like instructions and nutritional info.<br>
- **Favorites Management:** Save and remove favorite recipes (requires user authentication).<br>
- **Shopping List Generation:** Create and manage shopping lists from selected recipes.<br>
- **User Authentication:** Sign up, log in, and log out using Firebase Auth.<br>
- **Responsive Design:** Works on desktop and mobile devices.<br>
- **Real-Time Data:** Sync favorites and lists across sessions via Firebase Firestore.<br>
## Tech Stack
- **Frontend:** Vue.js 3, Vue Router, Vuex<br>
- **Backend/Services:** Firebase (Auth, Firestore), Spoonacular API<br>
- **Styling:** CSS (or integrate Vuetify for UI components)<br>
- **Other:** Axios for API calls, Node.js for development<br>
## Prerequisites
- Node.js (v16 or higher)<br>
- npm or yarn<br>
- Firebase account (for project setup)<br>
- Spoonacular API key (free tier available)<br>
